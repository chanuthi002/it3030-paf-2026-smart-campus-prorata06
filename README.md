📚 Smart Campus Operations Hub

This project is developed for the Programming Applications and Frameworks (IT3030) assignment at SLIIT. It is a full-stack web application designed to manage university resources, bookings, and maintenance operations.

🛠️ Technologies Used
Frontend: React.js

Backend: Spring Boot (REST API)

Database: MongoDB / SQL
